# Fullstack-Users - Client

## Description

Client frontend for Fullstack-Users. Based on [Angular](https://angular.io).

## Installation

```bash
$ npm install
```

## Building

```bash
$ npm run build
```

## Running the app

```bash
# development
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
# You first need to start the server
$ npm run test:e2e
```
